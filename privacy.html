Information We Collect

When you use our browser extension, we collect the following information:

Hashed (Pseudonymized) Data:

  • Disk Serial Number Hash
  • Motherboard Serial Number Hash
  • System Drive Volume Serial Number Hash
  • MAC Address Hash
  • Registered Account Hash
  • Installed Extensions List Hash
  • Full Font List Hash

All of these identifiers are processed using strong one-way hash algorithms. Hashed data cannot be used, either by us or by any third party, to directly identify you or reconstruct the original value.

Plaintext Data:

  • System Install Time
  • Persistent Device ID
  • Merchant Internal ID
  • Current URL（Only SHEIN sites）

Plaintext fields are collected only as required to ensure system integrity, identify your device consistently for security.

  • Further purposes may be described in the [Global Seller Privacy Notice] where applicable.

  • 主板序列号哈希值
  • 系统盘卷序列号哈希值
  • MAC地址哈希值
  • 注册账户哈希值
  • 已安装扩展列表哈希值
  • 完整字体列表哈希值

所有这些标识符均通过强一向哈希算法处理。哈希化的数据无法被我们或任何第三方直接用于识别您或还原原始值。

明文数据：

  • 系统安装时间
  • 持久设备 ID
  • 商家内部 ID

明文字段仅在确保系统完整性、持续识别您的设备以保障安全时被采集。
